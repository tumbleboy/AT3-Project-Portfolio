Weather App Development

I developed a weather application using CommunityToolkit.Mvvm, where I focused on creating a user-friendly interface and implementing API integration. This project allowed me to enhance my skills in app development while building a robust ViewModel to manage data binding and interactions with an external weather API.

Eyegaze Assistive Technology Proposal

As part of a collaborative team, I contributed to the development of Eyegaze, an assistive technology project designed to help individuals with disabilities access public services in the City of Joondalup. My role included researching accessibility needs, identifying key locations for implementation, and exploring tools provided by assistive technology providers like Link AT.
